# 🚚 Vendor Invoice Intelligence

A supervised machine learning project for intelligent vendor invoice analysis — including invoice anomaly flagging and freight cost estimation — deployed as an interactive Streamlit web dashboard.

---

## 📌 Project Overview

This project analyses vendor invoice data from a retail inventory database and applies machine learning to solve two business problems:

1. **Invoice Flagging** — Detect potentially anomalous invoices using a Random Forest Classifier trained on billing discrepancy and receiving delay features.
2. **Freight Cost Estimation** — Predict the shipping freight charge for an order using a Linear Regression model trained on quantity and order value features.

## 🛠️ Setup & Installation

### 1. Clone the Repository

```bash
git clone https://github.com/lakshmigowthami6/Vendor_Integrity_Tracker.git
cd Vendor_Integrity_Tracker
```

### 2. Install Dependencies

```bash
pip install -r deployment/requirements.txt
```

### 3. Add the Database

> ⚠️ The `data/inventory.db` SQLite database is excluded from this repository due to its large file size (~405 MB). You must provide your own copy.

Place the database at:
```
data/inventory.db
```

### 4. Train and Save the Models

Run the training script to generate serialized model binaries:

```bash
python deployment/train_and_save.py
```

This will generate the following files in `deployment/models/`:
- `invoice_flagging_model.joblib`
- `invoice_flagging_scaler.joblib`
- `freight_model.joblib`

### 5. Launch the Dashboard

```bash
streamlit run deployment/app.py
```

Or on Windows, double-click `deployment/run_dashboard.bat`.
